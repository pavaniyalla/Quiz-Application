🚀 Excited to share my latest project: **Java Quiz Game Application** 🎯

I developed this project during my internship at Global Quest Technologies, where I built an interactive quiz game using core Java concepts and Java Swing.

🔹 **Project Highlights:**
✔️ Console-based quiz game
✔️ GUI-based version using Java Swing
✔️ 10 Multiple Choice Questions (MCQs)
✔️ Lifelines (50-50, Audience Poll, Skip)
✔️ Reward system with safe levels
✔️ Instant feedback for answers

💡 This project helped me strengthen my understanding of:

* Arrays & Loops
* Conditional Statements
* Object-Oriented Programming
* Event Handling (Swing)
* User Interaction Design

🎮 The application simulates a real quiz show where users can answer questions, use lifelines strategically, and win rewards based on performance.

Java | Java Swing
